# Supermarket-Grocery-Sales-analytics-dashboard
Excel Data Analysis Project  on Retail Grocery Sales
## Project Overview
This project analyzes the retail grocery sales of a supermarket and the different factors affecting the sales using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power query editor for Data cleaning
- Pivot Charts
- Dashboard Design
- Slicers and Timeline
- KPI Cards

## Key Metrics
- Total Sales(revenue)
- Total Profit
- Profit Margin %
- Total Orders
- Average Discount
- Top Performing Category

## Insights
- Certain product categories generate the highest revenue.
- Higher discounts may increase order volume but tend to reduce profit margin.
- Some regions generate more revenue(sales) than other regions.Implementing better marketing and product promotion compaigns in the under performing regions would be useful.
